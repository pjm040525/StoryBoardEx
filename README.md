# Meeting Management App

모임 관리 애플리케이션 - 일정, 회비, 스토리를 한 곳에서 관리하세요.

원본 디자인: https://www.figma.com/design/g86HsN4ARkX3CJyVcjQLmK/Meeting-Management-App

## 주요 기능

- 📅 일정 관리 및 투표
- 💰 회비 관리
- 📖 스토리 공유
- 📊 통계 및 분석
- 👥 모임 관리

## 기술 스택

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Radix UI
- Lucide Icons

## 실행 방법

### 의존성 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 프로덕션 빌드
```bash
npm run build
```
